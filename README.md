Графическое приложение для управления задачами с поддержкой
регистрации пользователей, дедлайнов, категорий и истории выполненных задач.

Проект реализован с использованием **PyQt6** и **SQLAlchemy**,
данные хранятся в базе данных **PostgreSQL**.



## Возможности

- регистрация и авторизация пользователей;
- добавление задач с:
  - описанием;
  - категорией;
  - дедлайном (дата и время);
- просмотр текущих задач;
- отметка задач как выполненных;
- история выполненных задач;
- поиск задач:
  - по тексту
  - по диапазону дат дедлайна.


## Руководство по эксплуатации

При первом запуске необходимо зарегистрироваться, указав логин и пароль
Чтобы добавить задачу, нужно нажать на кнопку "Добавить задачу", предварительно указав ее описание. Далее всплывет дополнительное окно с выбором дедлайна и категории.

Если вы выполнили задачу, выберете ее левой кнопкой мыши и нажмите 'задача выполнена', она перейдет в блок 'история выполненных задач'

Если дедлайн задачи просрочится, рядом с таском появится сообщение 'ПРОСРОЧЕНО!'

Чтобы осуществить поиск по задачам, нужно выбрать промежуток времени окончания дедлайна и (по желанию) указать ключевое слово или подстроку


## Способы установки

### Требования

- Python **3.12.3** или выше  
- PostgreSQL **15** или выше  
- Docker **29.1.3** или выше

### Настройка Docker

1. Установите Docker, следуя [официальной инструкции](https://docs.docker.com/get-docker/).
2. После установки Docker, убедитесь, что он работает, выполнив в терминале команду:
   ```bash
   docker --version
   ```
3. Перейдите в директорию с клонированным репозиторием:
   ```bash
   cd <name>
   ```
4. Запустите Docker-контейнер с приложением, выполнив команду:
   ```bash
   docker compose up
   ```

### Установка приложения
1. Проверьте, что база данных запущена 
   ```bash
   docker ps
   ```
2. Клонировать репозиторий 
```bash
git clone https://github.com/<your-username>/<name>.git
cd <name>
```
3. Создайте виртуальное окружение и скачайте через pip необходимые модули из requierments.txt
```bash
python3 -m venv venv
source venv/bin/activate
```
4. Запустите приложение
```bash
python3 -m app.main
```
